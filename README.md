# FluidJobs.ai

A modern job platform with AI-powered features built with React and Node.js.

## 🚀 Quick Start

### Frontend (React)
```bash
cd FluidJobs.ai
npm install
npm start
```

### Backend (Node.js/Express)
```bash
cd backend
npm install
npm run dev
```
